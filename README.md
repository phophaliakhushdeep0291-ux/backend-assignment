Full-Stack Task Management System (MERN)
This project is a secure, scalable REST API built with Node.js, Express, and MongoDB, featuring a React.js frontend. It was developed as part of the Backend Developer Intern assignment for Primetrade.ai.

Backend (Primary Focus)
User Authentication: Secure Registration and Login with password hashing (bcrypt).

JWT Identity Management: Access and Refresh token logic for persistent sessions.

Email Verification: Token-based email verification system.

Role-Based Access Control (RBAC): Middleware to distinguish between User and Admin roles.

Task CRUD: Full Create, Read, Update, and Delete operations for user-specific tasks.

Security: Centralized error handling, input validation, and protected routes.

Frontend
React UI: Responsive interface for Auth and Task management.

Protected Routes: Dashboard access restricted to authenticated users.

Context API: Global state management for user authentication.
